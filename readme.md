# TESTES AUTOMATIZADOS PARA ERP REAL

## Sobre

O sistema é um conjunto extenso de testes para o sistema ERP REAL, da Mitis Tecnologia. O fluxo dos testes automatizados inclue a autenticação no sistema; execução de diversos testes em diferentes rotinas; gravação interativa dos testes que falharam; e log final, informando vários dados sobre a execução.
Também existem outros recursos úteis dos quais o sistema se aproveita, como a paralelização dos testes, emulação de dispositivos, variedade de navegadores, entre outros.

## (Instalação/pré-requisitos)











O sistema atual é a automatização de testes para o sistema ERP Real, da Mitis Tecnologia. Ele tem como objetivo reduzir o trabalho manual para garantir a qualidade do ERP; cobrir o máximo de áreas onde o sistema pode dar erro, com testes que serão rodados quantas vezes forem necessárias (no sentido de, alguém poderia deixar de fazer algum teste específico, enquanto que o sistema fará todos); passar um feedback mais rápido para os desenvolvedores se alguma funcionalidade foi "quebrada" com alguma atualização (através do Github Action). Todos esses objetivos vizam aumentar a qualidade do Real, com pouco esforço e tempo gasto pelos funcionários do setor de QA.
